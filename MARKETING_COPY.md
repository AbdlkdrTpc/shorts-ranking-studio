## Launch Post

I built a browser-based tool for making ranking videos faster.

You set the topic, add your items, reveal cards one by one, place them into the ranking, and export a clean 9:16 video.

It is made for creators producing content like:

- best perfumes
- best movies
- best footballers
- hottest takes
- top 5 / top 10 formats

Features:

- editable title and ending hook
- image and text cards
- shuffle reveal animation
- click / drag / keyboard placement
- mini camera bubble
- transparent mode and test backgrounds
- built-in 1080x1920 recorder

If you publish on X, LinkedIn, YouTube, or Instagram, this should save a lot of time.

## LinkedIn Post

I built a lightweight browser tool called Shorts Ranking Studio for creators producing ranking-based short videos and visual opinion content.

The idea was simple: reduce the friction between idea and upload.

With it, you can:

- define a topic and ending hook
- add text or image-based items
- reveal cards with a shuffle effect
- place them live into a ranking
- add webcam bubble and ambient effects
- export a clean 1080x1920 video

It is especially useful for recurring content formats where speed matters across X, LinkedIn, YouTube, and Instagram.
